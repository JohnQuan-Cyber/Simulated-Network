# The Simulated Network Project

By James, Johnny, and Devon

## About Our Project
The simulated network project aims to emulate a real network through software-defined networking on Proxmox.

## Goals
- Design a software-defined network that replicates a physical network.
- Implement all of the skills learned over the four semesters into the project.
- Use enterprise-level tools such as Proxmox and Cisco C8000V router images.
- Accurately define costs as if it were a physical network.

## What We Have Done So Far
So far, we have spent tens of hours designing and testing the network topology, tuning and testing virtual machines, and communicating with Cisco and instructors to acquire virtual router images.
