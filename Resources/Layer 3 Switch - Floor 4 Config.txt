==================== Layer 3 Switch - Floor 4 Configurations ====================

configure terminal

interface range GigabitEthernet 1/0/1-2
description Routing Ports
no switchport
exit

interface range GigabitEthernet 1/0/3-6
description Trunking Ports
switchport mode trunk
switchport trunk allowed vlan 10,20,30,40,50,60,70,80,90
no shutdown
